# Gatekeeper Bot

Telegram moderation bot that greets newcomers, enforces a tappable human verification, detects common spam, and provides admin controls with a rolling action log.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
